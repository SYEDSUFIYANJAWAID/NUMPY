# NUMPY
This repository contains my complete learning journey of NumPy, starting from basics and moving towards advanced concepts used in Data Science and Machine Learning.  

🚀 About This Project

NumPy is the foundation of Data Analysis, Machine Learning, and Scientific Computing in Python.
This repo is designed to build strong understanding step by step:

Basics of NumPy
Array operations
Mathematical functions
Advanced indexing & broadcasting
Real-world applications

📚 Topics Covered

🟢 Basics
What is NumPy?
Creating arrays (np.array, np.zeros, np.ones)
Array attributes (shape, ndim, dtype)
Basic indexing & slicing

🟡 Intermediate
Array reshaping (reshape, ravel)
Stacking & splitting arrays (np.hstack, np.vstack, np.split)
Mathematical operations (+, -, *, /)
Aggregation functions (sum, mean, min, max)

🔴 Advanced
Broadcasting rules
Fancy indexing
Boolean masking
Linear algebra basics (dot, matmul)
Random module (np.random)
Performance optimization in arrays
